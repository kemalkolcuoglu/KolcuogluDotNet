## Getting Started

**Requirements**

- [ASP.Net Core 3.1](https://dotnet.microsoft.com/)

**Quick Start**

``` bash
git clone https://github.com/kemalkolcuoglu/KolcuogluDotNet
cd KolcuogluDotNet/src/
dotnet run
```
